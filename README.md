## 🌌 解决什么问题？(The Solution)

在这个 AI 寒武纪大爆发的时代，你是否感到**“错失焦虑 (FOMO)”**？
每天都有新工具、新模型、新概念。你担心如果不时刻盯着 Twitter 就会被时代抛弃。

**Nexus 是你的定心丸。**
我们提供一张**“全知动态地图”**。社区与 AI 智能体 24 小时监控全球技术动态，将最新的、真正有价值的技术（如 Gemini Pro, Sora, Agent 搭建）实时更新到地图上。

**你不需要满世界找教程。只要地图上的新节点亮了，就说明它值得学。在这里，你永远不会错过未来。**

## 🌟 核心特性

* **全知动态地图 (The Living Map)**: **治疗资讯焦虑。** 这是一个活的知识图谱。当新的一项颠覆性技术发布，我们的地图会自动生长出新的分支节点。
* **多元模态学习 (Multi-modal)**:
    * **冒险模式 (Game)**: 亲手搭建 Agent，在代码沙盒中实战。
    * **传说模式 (Story)**: 在科幻/悬疑小说中理解复杂的算法原理。
    * **剧院模式 (Video)**: 观看最新的实操精讲视频，带实时问答。
    * **图馆模式 (Archive)**: 快速查阅结构化思维导图。
* **全民技能覆盖**: 从 AI 办公流到无人机维修，从 K12 基础到职业进阶，覆盖生存与发展的方方面面。
* **知识平权**: 打破信息差，让每个人都能第一时间接触最前沿的生产力工具。
* **共进社区**: 实时“酒馆”交流，路书分享，师徒系统。

## 🚀 快速开始 (Quick Start)

### 环境要求
* Node.js >= 18.0.0
* npm 或 yarn

### 本地运行
```bash
# 1. 克隆仓库
git clone [https://github.com/your-username/nexus-academica.git](https://github.com/your-username/nexus-academica.git)

# 2. 进入目录
cd nexus-academica

# 3. 安装依赖
npm install

# 4. 启动开发服务器
npm run dev
```

打开浏览器访问 `http://localhost:5173` 即可看到 Nexus 原型。
