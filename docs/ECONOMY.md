# Nexus Economic Protocol (NEP): 经济与激励白皮书

**版本**: 1.1 (Writers Included)

> **"Code implies Consent, Contribution implies Ownership."**

在 Nexus Academica，我们不雇佣员工，我们拥有合伙人。

---

## 1. 核心商业闭环 (The Profit Loop)
1. **创造**: 社区构建内容。
2. **体验**: 学习者付费/使用。
3. **收入**: 订阅费、硬件利润、企业认证。
4. **分红**: 利润流入贡献者。

## 2. 收入来源 (Inflow)
### 2.1 Nexus Pass 订阅 (40%)
* $9.9/月，解锁高级副本、AI 导师、云算力。

### 2.2 物理工坊 (Hardware) (30%)
* O2O 硬件套件销售（无人机、电子积木）。

### 2.3 认证与人才输送 (B2B) (30%)
* 企业付费猎头费，基于链上技能认证。

## 3. 贡献者收益分配 (Distribution)
**532 分配原则**（净利润）：

* **50% - 社区分红池**: 创作者。
* **30% - 核心开发池**: 开发者。
* **20% - 运营与储备金**: 运营。

### 3.1 创作者 (Creators) - "Design-to-Earn"
$$	ext{收益} = (	ext{有效通关人数} 	imes 0.7) + (	ext{用户好评率权重} 	imes 0.3)$$
* **长尾收入**: 持续进账。
* **多模态激励**: 
    * **小说家**: 根据“阅读完读率”和“收藏数”计算权重。
    * **视频创作者**: 根据“完播率”计算权重。
    * **游戏设计师**: 根据“通关率”计算权重。
    * **路书整理者**: 优秀的“路书”整理者也可获得收益。

### 3.2 开发者 (Developers) - "Commit-to-Earn"
* **RPGF 机制**: 根据 PR 积分，季度分红。
* **技术股**: 核心维护者期权。

### 3.3 硬件生态伙伴
* 获得硬件利润的 15%-25% 版税。

## 4. 信任与透明 (Transparency)
1. **开源账本**: 流水实时公开。
2. **智能合约**: 未来链上自动分账。
